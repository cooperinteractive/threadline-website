Executive Summary

Greenfield

PullSheet enters a greenfield market. No production-grade visual lookbook builder exists today. Current workflows are manual: printed photos, corkboards, scissors, and tape. The closest digital competitor (Dress Assistant) lacks production awareness, AI features, and PDF export. Canva and Pinterest offer general-purpose tools but have no connection to wardrobe inventory, no background removal, and no understanding of film production workflows.

PullSheet has a built-in distribution advantage through the existing THREADLINE user base. As an add-on module (not standalone), it benefits from zero customer acquisition cost for existing subscribers and creates a natural upsell path that increases THREADLINE's lifetime value per user.

Key Insight

No competitor combines: (1) production inventory integration, (2) AI background removal, (3) freeform canvas editing, and (4) professional PDF export. PullSheet owns this intersection entirely.

Pricing Rationale

  • Background removals are the primary cost driver (Clipdrop API usage). Tiering by removal count aligns price with actual infrastructure cost.
  • Export is cheap (client-side PDF generation), so it is unlimited on all paid tiers.
  • Team size aligns with production scale. Individual costumers need 1-3, department heads need up to 10, studios need unlimited.
  • Free tier during beta builds the user base and generates feedback before paid launch.
  • $4.99 Essential tier is low enough for individual adoption without department budget approval.
  • $24.99 Production tier captures studio-scale value while remaining far below enterprise SaaS pricing.

Margin Note

PullSheet's cost structure is heavily weighted toward the Clipdrop API for background removals. All other costs are either shared with THREADLINE (Supabase) or zero-incremental (client-side PDF generation). This means gross margins improve significantly as the user base grows and per-unit API costs decrease with volume discounts.

Distribution Strategy

4 Channels
Primary Channel

In-App Promotion (THREADLINE)

"Build a lookboard" CTA surfaces naturally within THREADLINE's inventory and talent views. Zero acquisition cost. Users discover PullSheet in the context where they already need it.

Secondary Channel

App Store Discovery

Organic App Store search for "lookbook builder," "wardrobe lookboard," and "costume design." ASO-optimized listing with before/after screenshots and demo video.

Tertiary Channel

Social Media + Industry Events

Instagram, LinkedIn, and Facebook content showcasing lookboard transformations. Presence at IATSE events, Costume Designers Guild meetings, and film industry conferences.

Organic Channel

Deep Linking + Word-of-Mouth

THREADLINE surfaces PullSheet lookboards in project views, creating natural discovery. Wardrobe departments are tight-knit communities: one power user drives 5+ referrals per year.
