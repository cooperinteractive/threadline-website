02 / SVG Source
SVG Monogram
The PS monogram as scalable vector graphics. Use this SVG directly for high-resolution contexts or as a favicon.svg in modern browsers.
SVG Source Code
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 512 512">
  <defs>
    <linearGradient id="bg"
      x1="0%" y1="0%"
      x2="100%" y2="100%">
      <stop offset="0%"
        stop-color="#5A4228"/>
      <stop offset="50%"
        stop-color="#8B6F52"/>
      <stop offset="100%"
        stop-color="#C4946A"/>
    </linearGradient>
  </defs>
  <!-- Rounded rect background -->
  <rect width="512" height="512"
    rx="64" fill="url(#bg)"/>
  <!-- PS monogram -->
  <text x="256" y="300"
    text-anchor="middle"
    font-family="Inter, sans-serif"
    font-weight="300"
    font-size="220"
    fill="rgba(255,255,255,0.92)"
    letter-spacing="10">PS</text>
</svg>
03 / Implementation
Implementation Guide
Add these link tags to the <head> of pullsheetapp.io to register all favicon sizes. Place the downloaded PNG files in the root directory.
Size Purpose HTML Tag
favicon.svg Modern browsers (scalable) <link rel="icon" type="image/svg+xml" href="/favicon.svg">
16x16 Classic browser tab <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png">
32x32 Retina browser tab <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
48x48 Windows taskbar shortcut <link rel="icon" type="image/png" sizes="48x48" href="/favicon-48.png">
180x180 Apple touch icon (iOS) <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
192x192 Android home screen <link rel="icon" type="image/png" sizes="192x192" href="/android-chrome-192.png">
512x512 PWA manifest icon Referenced in manifest.json under "icons"
manifest.json snippet
{
  "icons": [
    {
      "src": "/android-chrome-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "/icon-512.png",
      "sizes": "512x512",
      "type": "image/png",
      "purpose": "any maskable"
    }
  ]
}
