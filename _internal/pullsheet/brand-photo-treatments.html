01 / Technique
The Ghosted Photo Technique
A distinctive visual treatment that places garment photography at extremely low opacity behind content, creating atmospheric depth without competing with text.

The ghosted photo technique takes high-quality garment photography and renders it as a barely-visible background layer. The image becomes texture rather than content, adding visual richness and brand recognition without demanding attention.

Three CSS properties work together to achieve this effect: opacity controls visibility (0.06 to 0.12 range), filter: grayscale(1) strips color to prevent palette conflicts, and mix-blend-mode controls how the ghost composites with its background.

The technique works best on solid or gradient backgrounds. On light backgrounds (cream, bone, white), the ghost appears as warm shadow. On dark backgrounds (near-black, void), use mix-blend-mode: overlay so the ghost appears as a subtle light impression.

Keep opacity under 0.12 at all times. The moment a ghost image becomes "readable" as an actual photograph, it has gone too far. The viewer should feel the presence of fashion imagery without consciously registering it.

Core CSS Properties

opacity 0.06 - 0.12
filter grayscale(1)
mix-blend-mode overlay | normal
object-fit cover
position absolute
pointer-events none
background (light) var(--cream)
background (dark) var(--near-black)

Six Treatment Variations
Each treatment adapts the ghosted technique for a different layout context. Use the CSS snippets below as starting points for implementation.

Treatment 01
Feature Card
Cream background with a garment silhouette ghosted at 0.08 opacity. Text sits at the bottom of the card, reading clearly over the barely-there image.
Use case: Feature announcements, marketing cards, app store screenshots, and any card that needs atmospheric depth without a real photograph. 
/* Feature Card ghost treatment */
.card {
  background: var(--cream);
  position: relative;
  overflow: hidden;
}
.card img.ghost {
  position: absolute;
  inset: 0;
  width: 100%;
  height: 100%;
  object-fit: cover;
  opacity: 0.08;
  filter: grayscale(1);
  pointer-events: none;
}

Treatment 06
Bento Grid
Multi-cell grid where each cell uses a different background color and ghost opacity. The left cell spans two rows. Each cell gets its own ghost treatment appropriate to its background tone.
Use case: Feature overview sections, app store marketing assets, dashboard-style layouts, and landing page feature grids. 
/* Bento Grid ghost treatment */
.bento {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 1fr 1fr;
  gap: 1px;
}
.bento-cell {
  position: relative;
  overflow: hidden;
}
/* Light cell */
.cell-light img.ghost {
  opacity: 0.09;
  filter: grayscale(1);
}
/* Accent cell */
.cell-accent img.ghost {
  opacity: 0.06;
  mix-blend-mode: overlay;
}
/* Dark cell */
.cell-dark img.ghost {
  opacity: 0.08;
  mix-blend-mode: overlay;
}
03 / Direction
Photography Direction
The ghost technique is only as good as the source photography. PullSheet imagery should reflect the professional wardrobe world, not generic stock photography.

Photograph These

  • Lookboard compositions (garments arranged on a surface or board)
  • Flat-lay arrangements of curated outfit groupings
  • Wardrobe racks with garments hanging in sequence
  • On-set wardrobe details (steamer, tags, fitting room moments)
  • Fabric textures and material close-ups
  • iPad or tablet displaying PullSheet interface
  • Behind-the-scenes production wardrobe setups
  • Garment details: labels, stitching, hardware

Never Use

  • Generic stock photography of any kind
  • Posed model photography (this is a tool, not a lookbook)
  • Lifestyle imagery unrelated to wardrobe work
  • Low-resolution or poorly lit source images
  • Images with visible watermarks or third-party branding
  • Cluttered or messy compositions (even when ghosted, chaos reads)
  • Competitor product screenshots or UI references
  • AI-generated photography (authenticity matters)
